Starter plugin for Tobwai project.

== Description ==
Tobwai WordPress Plugin is a minimal starter template with admin settings, i18n, activation/deactivation hooks, and uninstall.

== Installation ==
1. Upload the `tobwai-wordpress-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure via **Tobwai Plugin** menu.
